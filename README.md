# java-bibliotecar
CRUD приложение Spring MVC, PostgreSQL (jdbctemplate), Thymeleaf, Tomcat

Приложение умеет:
* добавлять читателей и книги
* назначать книги читателям
* следить за читателями и наличием книг у них
